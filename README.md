# Operations Performance Tracker

A Python-based workforce productivity tracking and KPI monitoring system 
developed to optimize customer service operations and improve team efficiency.

## Project Overview

This project was built to support real-world customer operations management.
It tracks activity, logs performance data, and generates structured datasets
for KPI analysis and reporting.

The system was used in a high-volume service environment managing:

- 200+ cases per day
- 98% performance evaluation score
- Reduced resolution time
- Supervised a team of 7 agents

## Key Features

- Active window tracking for workflow monitoring
- Automated time logging
- CSV performance export
- Data-ready structure for Power BI dashboards
- Designed for KPI tracking & operational analytics
